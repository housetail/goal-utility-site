# 工具网站规划

## 网站目标
- 提供 3 款可直接在浏览器中使用、具有高实用价值的工具。
- 为广告展示预留位置并集成主流广告脚本。
- 通过免费托管平台（推荐 Cloudflare Pages 或 Vercel）上线。

## 页面结构
1. **导航区**：展示品牌信息与工具列表快捷入口。
2. **英雄横幅**：强调网站价值主张、引导用户开始使用。
3. **工具区块**：包含 3 款工具，每个工具独立卡片，并内含介绍、操作表单与结果展示。
4. **广告区块**：顶部横幅 + 工具区侧边直式广告位，便于第三方广告填充。
5. **CTA 底部区**：引导订阅邮件或保存网址。

## 工具设计
1. **全球汇率换算器**
   - 功能：支持 150+ 全球货币之间的实时换算。
   - 数据源：`https://api.exchangerate.host/latest` 免费 API。
   - UI：货币选择下拉、输入金额、即时显示结果与更新时间。

2. **房贷月供计算器**
   - 功能：根据贷款总额、利率、年限计算等额本息月供、总利息。
   - 数据源：前端公式计算，无需后台。
   - UI：三个输入框 + 展示结果卡片。

3. **旅行预算规划器**
   - 功能：根据旅行天数、同行人数、每日预算建议生成整体预算与人均费用。
   - UI：输入基本参数 + 可选应急金比例，输出详细预算。

## 技术栈
- 纯前端静态站点：`HTML + CSS + JavaScript`。
- 无需构建工具，易于在任意静态托管平台部署。

## 广告集成方案
- 默认嵌入 Google AdSense 代码片段。
- 预置 `data-ad-client` 和 `data-ad-slot` 占位符，部署前需替换为实际值。
- 若使用国内广告联盟，可将脚本替换为自定义 `<script>` 与广告容器。

## 部署方案建议
- **首选**：Cloudflare Pages
  - 免费、支持静态站点、全球加速。
  - 只需连接 Git 仓库或上传 ZIP。
- **备选**：Vercel / Netlify
  - 同样免费部署静态资源，配置简单。

## 后续任务
- 编写页面 `index.html`、样式 `styles.css`、脚本 `main.js`。
- 在工具函数上补充注释、确保响应式布局。
- 编写 `DEPLOYMENT.md` 指南，包含部署与广告接入步骤。
